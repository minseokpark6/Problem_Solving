<h5>문제</h5>

<p><code>FOOD_PRODUCT</code> 테이블에서 가격이 제일 비싼 식품의 식품 ID, 식품 이름, 식품 코드, 식품분류, 식품 가격을 조회하는 SQL문을 작성해주세요. </p>

<hr>

## 💡Solving
```sql
select product_id, product_name, product_cd, category, price
from food_product
order by 5 desc
limit 1;
```

<br />

## 💡Solving_Sub Query
```sql
select product_id, product_name, product_cd, category, price
from food_product
where price = (select max(price) from food_product);
```
